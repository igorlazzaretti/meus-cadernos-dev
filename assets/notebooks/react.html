React

React é uma biblioteca JavaScript criada pelo Facebook, utilizada para construir interfaces de usuário (UIs) de forma eficiente e declarativa. Lançada em 2013, ela permite o desenvolvimento de componentes reutilizáveis, que gerenciam seu próprio estado, facilitando a criação de aplicações web dinâmicas e complexas.
