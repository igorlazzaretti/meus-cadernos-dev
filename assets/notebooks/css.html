Breve Resumo

Cascading Style Sheet e é usado para estilizar elementos escritos em uma linguagem de marcação como HTML. O CSS separa o conteúdo da representação visual do site. CSS foi desenvolvido pelo W3C (World Wide Web Consortium) em 1996. A relação entre HTML e CSS é bem forte. Como o HTML é uma linguagem de marcação (o alicerce de um site) e o CSS é focado no estilo (toda a estética de um site), eles andam juntos.

Melhores práticas: crie uma pasta Style ou CSS dentro de Assets. Lá crie seu arquivo stylesheet.css
