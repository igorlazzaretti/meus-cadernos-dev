Breve Resumo

JavaScript foi criado por Brendan Eich em 1995 enquanto ele trabalhava na Netscape Communications Corporation. A ideia inicial era desenvolver uma linguagem que pudesse ser usada tanto por designers quanto por programadores, para criar páginas web dinâmicas e interativas. Originalmente, a linguagem foi chamada de Mocha, depois renomeada para LiveScript, e finalmente JavaScript, em uma jogada de marketing para capitalizar na popularidade do Java na época.

JavaScript rapidamente se tornou uma parte essencial do desenvolvimento web, sendo implementado em todos os principais navegadores da época. Em 1997, o JavaScript foi padronizado pela Ecma International, resultando na criação da especificação ECMAScript. Desde então, a linguagem tem evoluído continuamente com novas versões do ECMAScript, introduzindo funcionalidades avançadas e melhorias de performance.

No contexto do front-end, JavaScript é utilizado para manipular o DOM (Document Object Model), permitindo a atualização e alteração de conteúdo sem a necessidade de recarregar a página. Com o tempo, várias bibliotecas e frameworks, como jQuery, Angular, React e Vue.js, foram desenvolvidos para facilitar a criação de interfaces de usuário ricas e responsivas. Essas ferramentas ajudaram a solidificar JavaScript como uma linguagem vital para o desenvolvimento web moderno, possibilitando experiências interativas e dinâmicas em navegadores de internet.

Grandes empresas usam: Google, Facebook, Netflix, Linkedin, Amazon e Twitter por exemplo usam.

O que são Variáveis

É uma parte de memória separada para contêr alguma informação.

var, let, const

var: Usado na versão mais antiga de JavaScript, tem escopo global ou de função.

let: Introduzido no ES6 (ECMAScript 2015), tem escopo de bloco, o que significa que só é acessível dentro do bloco em que foi declarado.

const: Também introduzido no ES6, é usado para declarar variáveis cujos valores não podem ser alterados depois de definidos. Assim como let, tem escopo de bloco.

Dicas Rodolfo, var não usa mais. Usa sempre const, se precisar mudar seu info, usa então let.

Em JavaScript, um array é uma estrutura de dados que permite armazenar múltiplos valores em uma única variável. Os arrays podem conter qualquer tipo de dado, como números, strings, objetos e até outros arrays. Eles são ordenados, o que significa que cada elemento tem um índice numérico, começando do zero.

DOM e como Manipulá-la

O DOM (Document Object Model) é uma interface de programação para documentos HTML e XML. Ele representa a estrutura do documento como uma árvore de nós, onde cada nó é um objeto que corresponde a uma parte do documento. Com o DOM, você pode manipular o conteúdo, estrutura e estilo de documentos web de forma programática usando JavaScript.

Estrutura do DOM

Documento: O objeto raiz que representa o documento completo.

Documento: O objeto raiz que representa o documento completo.

Elementos: Representam tags HTML e podem conter atributos, texto e outros elementos.

Atributos: Representam atributos HTML e são encontrados dentro das tags de elementos.

Texto: Nós que contêm o texto dentro de um elemento.
